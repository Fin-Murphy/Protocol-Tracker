Task unifier project
ultimate goal is for Protocol Tracker to pull in To-dos from disparate applications 
like Gcal, Canvas, Gmail, Notion, Obsidian, Slack, etc. and make a unified list all 
the tasks you have to do in a day. This way you don't need to spend time bobbling 
between different applications, maintaining multiple tabs, and switching between apps
all the time, which will significantly reduce scheduling overhead and dramatically 
increase productivity
